Base location.
The root folder is the project directory.

First level folders.
- Code: Python processing and analysis scripts.
- Files: Paper documents and main tables.
- Processed_Data: Final processed databases.
- Raw_Data: Original unprocessed data.
- Results: Analysis outputs, tables and reports.

First level files.
- Run_All.py: Pipeline orchestrator script.
- Requirements.txt: Python dependencies.
- Readme.md: This document.

Code detail.
- Code/Clustering.py: Participant clustering.
- Code/Correlation_Analysis.py: Correlations between variables.
- Code/Data_Processing.py: Raw data processing.
- Code/Descriptive_Analysis.py: Descriptive statistics.
- Code/Models.py: Robust, GLM and logistic models.
- Code/Segmented_Population_Analysis.py: Analysis by segments.
- Code/Statistical_Analysis.py: Statistical tests.

Files detail.
- Files/Paper.docx: Paper document (editable).
- Files/Paper.pdf: Paper document (PDF).
- Files/Supplementary material.pdf: Supplementary material.
- Files/Tables.xlsx: Main paper tables.
